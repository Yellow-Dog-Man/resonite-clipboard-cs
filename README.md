# resonite-clipboard-cs

This repository contains the C# bindings for the [resonite-clipboard-rs](https://github.com/Yellow-Dog-Man/resonite-clipboard-rs) library, taken from the community project [ResoniteLinuxClipboard](https://github.com/GrandtheUK/ResoniteLinuxClipboard).
